# Dynamic Stability Derivatives using CFD

Estimation of aircraft dynamic stability derivatives using a CFD-based forced oscillation method.

## Method
The aircraft model undergoes controlled pitching oscillations using dynamic mesh in ANSYS Fluent.

## Derivatives Estimated
- CL_alpha
- CL_alpha_dot
- Cm_alpha
- Cm_q

## Application
Solar HALE UAV and SACCON reference model.
